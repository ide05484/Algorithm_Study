# 스터디 규칙
### 1. 파일 올릴 때 파일 명
```
[년월일] 문제 제목_문제번호
ex) [240324] 영화감독 숌_1436
```
### 2. commit message
```
[백준] 문제제목_문제번호
[SWEA] 문제제목_문제번호
```
### 3. 규칙
* 2월 : IM 준비
* 3월~ : class 2부터 차근차근 풀기
  * 브론즈 단계 : 매일 풀기
  * 실버 단계 : 이틀에 한 번
